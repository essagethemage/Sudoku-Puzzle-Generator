3. Short description of the project: This program is designed to produced a solved 9x9 Sudoku Puzzle. Through recursive designs, this program attempts to find solutions that ensures that every column, row, and 3x3 boxes repeat the number 1-9 only once. However, it is important to notes that this program is not able to implement user inputs!

6. Summary of how the Sudoku board is generated: The boards starts off as an empty 9x9 grid. However, the first row is randomized to hold a unique vlaue from 1-9. Then, it check for empty cells within the board. Once it finds a empty cell, it attempts to plug in every value from 1-9 until it finds a vlaue that ensures no number is repeated in the respective column, row, or 3x3 box.
